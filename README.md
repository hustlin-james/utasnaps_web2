#UTASnaps

## Install
1. "npm install"
2. "bower install"

## E2E Testing
1. "grunt test"

## Notes:
### adding bower dependencies: 
1. add the name and version to the bower.json
2. do "bower update";
3. do "grunt wiredep" to automatically add the script tags to the index.html
